Heart Sensor System - 413 Final Project

This website will allow a user to sign-up or login to their account, add a device, change the settings, and most importantly, view the data in daily and weekly graphs.
The user's device will periodically ask them to take a pulse and blood oxygen saturation level reading, which will be requested every 30 minutes from 6am to 10pm.
The user is notified by a blinking light when a test should be taken, and will stop blinking when the test is completed.
